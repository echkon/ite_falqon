# Data for "Imaginary-time-enhanced feedback-based quantum algorithms for universal ground-state preparation"

This repository contains the numerical data and plotting scripts associated with the paper:

**Imaginary-time-enhanced feedback-based quantum algorithms for universal ground-state preparation**  
*Thanh Nguyen Van Long, Lan Nguyen Tran and Le Bin Ho*  

---

## Contents


- `fig1` – Data and python plot for fig1.
- `fig2` – Data and python plot for fig2.
- `fig3` – Data and python plot for fig3.
- `fig4` – Data and python plot for fig4.

---

## Requirements

To reproduce the plots, install:

- Python ≥ 3.9  
- [NumPy](https://numpy.org/)  
- [Matplotlib](https://matplotlib.org/)  
- [Pandas](https://pandas.pydata.org/)

---

## Usage

1. Clone or download this repository.  
2. Install the required Python packages.  
3. Run the plotting scripts
